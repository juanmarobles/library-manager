## Descripción general del sistema

Este proyecto consiste en el desarrollo de un **Sistema de Biblioteca**, cuyo objetivo es gestionar los préstamos y devoluciones de libros de manera sencilla y eficiente.

El sistema contempla las siguientes funcionalidades:

* Registro de usuarios.
* Registro y administración de libros.
* Préstamo de libros.
* Devolución de libros.
* Control de disponibilidad de ejemplares.
* Cálculo de multas simples por demora en la devolución.
* Estadísticas sobre:

  * Libros más solicitados.
  * Cantidad de préstamos realizados.

El proyecto fue desarrollado en **Python** utilizando programación orientada a objetos, buscando una estructura modular y de fácil mantenimiento.

## Requisitos

Antes de ejecutar el proyecto es necesario tener instalado:

* Python 3.10 o superior
* Git (opcional, para clonar el repositorio)

---

## Instrucciones de ejecución

### 1. Clonar el repositorio

```bash
git clone <URL_DEL_REPOSITORIO>
```

### 2. Ingresar al proyecto

```bash
cd <NOMBRE_DEL_REPOSITORIO>
```

### 3. Ejecutar la aplicación

Desde la consola ejecutar:

```bash
python main.py
```

Si tu sistema utiliza el comando `python3`:

```bash
python3 main.py
```
